我们的理念

Aurora Shop 是一个纯前端静态多页面示例站点，强调现代化 UI 与完整的交互演示：轮播、选项卡、无缝滚动、侧边栏、广告栏、 购物车与订单（localStorage）、留言板（DOM 动态增删改）、音视频控制等。

常见问题（FAQ）
Q：站点为何是多页面？
A：便于展示导航结构、目录规划与不同页面特效的组合。
结构
Q：购物车数据保存在哪里？
A：保存在浏览器 localStorage（演示），关闭浏览器后仍可保留。
存储
Q：为什么没有真实支付？
A：静态站点不接入后端，结算页仅生成订单并清空购物车。
演示
Q：音视频为什么可能无法自动播放？
A：浏览器通常禁止非用户手势触发的自动播放，需要点击播放按钮。
浏览器
Q：页面整体如何保持统一风格？
A：使用全站 CSS 变量、玻璃拟态卡片、统一按钮与表格样式，并由 JS 注入公共头尾。
UI
